## Synopsis
<b>`ECHO message`</b><br>
This command outputs the given `message`.

## Return Value
Returns the message.

## Examples

You can do this as shown below.

```sh
$ ECHO "Yuga Yuga"
```

```
"Yuga Yuga"
```
