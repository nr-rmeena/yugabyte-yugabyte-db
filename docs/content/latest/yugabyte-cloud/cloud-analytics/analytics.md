The **Analyze** tab allows you to create custom charts of cluster metrics.

![Cloud Analytics Analyze tab](/images/yb-cloud/cloud-analytics-analyze.png)

Metrics

- Operations per Second
- Latency
- CPU
- Memory
- System Load
- Disk Bytes/Sec
- Network Errors
- Overall RPCs
- Average SS Tables
